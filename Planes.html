Nuestros Planes

Este es el listado de nuestros planes básicos, una vez se haga contacto entre el coach y el usuario este podrá tener una experiencia mas específica y personalizada.

Plan 1: Analisis de partidas

En este plan tu coach te ayudara a repasar y estudiar tu estilo de juego mirando tus partidas, algunas que elijas vos y otras que elija el, este plan es ideal para jugadores que les cuesta encontrar sus errores.

Plan 2: Aprendiendo conceptos basicos

En este plan tu coach y vos van a estudiar los fundamentos basicos del juego como lo son el farm, items, objetivos y mas, ideal para jugadores con menos de 2 años de experiencia.

Plan 3: Perfeccionando toma de decisiones

En este plan, el cual consideramos el mas completo, tu coach llevara tu entendimiento del juego al maximo, proponiendo situaciones y comparandolas con jugadas del entorno profesional, el objetivo de este plan es que comprendas lo importante que son las decisiones por mas pequeñas que sean. Se recomienda un rango minimo de platino para este plan.

Plan 4: Estudiando temas personalizados

Este plan esta especialmente planeado para jugadores que sienten que fallan en un tema especifico o que se sienten estancados, este plan ayudara a revisar el estilo de juego propio para lograr encontrar el punto a pulir. Tambien se recomienda este plan para jugadores por encima del rango oro.
